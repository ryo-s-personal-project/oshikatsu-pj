# 推し活 フロントエンド

TypeScript + React (Vite) で構築した推し活支援システムのフロントエンドです。

## 機能

- **ログイン** … ユーザー名・パスワードでログイン（JWT）
- **ダッシュボード** … ログイン後のホーム。推しグループ・推しメンバーへの導線
- **推しグループ** … グループ名/会社名での検索、新規作成・編集
- **推しメンバー** … プレースホルダー（今後実装）

## 必要環境

- Node.js 18+
- バックエンド API（`backend/oshikatsu`）が起動していること（デフォルト: `http://localhost:8080`）

## セットアップ

```bash
npm install
```

環境変数で API のベース URL を変更する場合は、プロジェクトルートに `.env` を作成し:

```
VITE_API_BASE_URL=http://localhost:8080
```

（未設定時は上記がデフォルトです。`.env.example` を参照。）

## 開発サーバー

```bash
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。バックエンドは別ターミナルで起動してください。

## ビルド

```bash
npm run build
```

`dist/` に本番用ビルドが出力されます。

## プロジェクト構成（抜粋）

- `src/api/` … 認証・推しグループなどの API 呼び出し
- `src/components/` … 共通コンポーネント（レイアウト、エラー表示、認証付きルート）
- `src/contexts/` … 認証コンテキスト（ログイン状態）
- `src/lib/` … API クライアント、バリデーション（Zod）
- `src/pages/` … 画面コンポーネント（ログイン、ダッシュボード、推しグループ、推しメンバー）
- `src/types/` … API 用型定義

バックエンドの DTO および `GlobalExceptionHandler` のエラー形式に合わせた型・バリデーション・例外処理を入れています。
